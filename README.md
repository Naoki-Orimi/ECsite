## 本プロジェクトに関して

簡易型
ECサイト

登録した商品を購入→決済→発送
まで行えるようにする。

## アピール機能
- ログイン機能
- ユーザー登録
- 商品登録
- デザイン（bladeとTailwindCssを使っていく）


## ログイン機能
- Laravel Breeze
  - ログイン、登録、パスワードのリセット、電子メール検証、パスワード確認を含む、Laravel のすべての認証機能の最小限でシンプルな実装を可能とする

```
composer require laravel/breeze --dev

php artisan breeze:install react
```
